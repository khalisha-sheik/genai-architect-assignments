# Personalized Course Recommendation Engine

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Configure API Key

```bash
export GOOGLE_API_KEY="your_api_key"
```

## Run Project

```bash
python3 course_recommendation_engine.py
```

## Run MLflow

```bash
mlflow ui
```

Open:

http://127.0.0.1:5000
